## Project Overview

This project was completed as part of Week 1 of the White Band Associates Summer Internship Program. The objective of this task was to understand fundamental networking concepts and perform basic network analysis using Windows and Kali Linux.

---

## Objectives

* Identify network configuration details.
* Understand IP addressing and MAC addressing.
* Perform network connectivity testing.
* Analyze network paths using Traceroute.
* Create a basic network diagram.
* Document findings with screenshots and observations.

---

## Tools Used

* Windows 11
* Command Prompt
* VMware Workstation
* Kali Linux
* Canva
* GitHub

---

## Activities Performed

### Network Configuration Analysis

* Collected network information using `ipconfig /all`.
* Identified Hostname, IPv4 Address, MAC Address, Default Gateway, DNS Server, and Subnet Mask.

### Connectivity Testing

* Performed Ping Test using `ping google.com`.
* Verified successful communication with a remote server.

### Route Analysis

* Performed Traceroute using `tracert google.com`.
* Observed the network path and intermediate hops.

### Kali Linux Verification

* Verified network configuration using `ip addr`.
* Performed Ping and Traceroute tests from Kali Linux.

### Network Diagram

* Created a basic network diagram representing communication between devices and the Internet.
